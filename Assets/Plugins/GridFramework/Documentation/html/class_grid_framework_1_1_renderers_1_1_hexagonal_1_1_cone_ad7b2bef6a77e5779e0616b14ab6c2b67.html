This is an abstract method, the implementations are up to the subclasses. This method should be called before the array is allocated. note that both from and to are references, this allows us to convert them into a common format that can be used in the subsequent calculations. For example, absolute world dimensions could be converted to relative grid dimensions and all other calculations would only need to be implemented for relative grid dimensions.

It is advised not to call this method directly, rather call the UpdatePoints method which will ensure the proper order of calls.
