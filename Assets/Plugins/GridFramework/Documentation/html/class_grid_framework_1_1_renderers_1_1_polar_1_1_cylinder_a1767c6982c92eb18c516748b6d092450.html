int GridFramework.Renderers.Polar.Cylinder.Smoothness
getset

Smoothness of the grid segments.

Unity's GL class can only draw straight lines, so in order to get the sectors to look round this value breaks each sector into smaller segments. The number of smoothness tells how many segments the circular line has been broken into. The amount of end points used is smoothness + 1, because we count both edges of the sector.
