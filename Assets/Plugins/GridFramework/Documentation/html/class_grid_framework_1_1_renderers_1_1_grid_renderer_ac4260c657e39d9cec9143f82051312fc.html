Material GridFramework.Renderers.GridRenderer.Material
getset

You can use you own material if you want control over the shader used, otherwise a default material with the following shader will be used:

Shader "GridFramework/DefaultShader" {
SubShader {
Pass {
Blend SrcAlpha OneMinusSrcAlpha
ZWrite Off Cull Off Fog {
Mode Off
}
BindChannels {
Bind "vertex", vertex Bind "color", color
}
}
}
}

The shader itself can be find among the shaders as GridFramework/DefaultShader.
