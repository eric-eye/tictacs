Detailed Description

The manager is completely closed data-wise and only accessible through its public methods. You should not have to use the manager yourself, renderers will register and unregister themselves when they are enabled or disabled.

However, if for whatever reason you still want to access the manager you can safely register and unregister renderers, there will be no duplicates and already unregistered renderers will be ignored.
